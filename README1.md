# 🚀 My Token Launcher Standard

A web application to easily create and launch tokens on the **Aptos blockchain**.

---

## 🧩 What is this?
This is a token launcher built for the Aptos ecosystem. It allows users to:
- Create custom tokens on Aptos.
- Deploy tokens directly from the UI.
- Interact with smart contracts for minting and managing tokens.

---

## 🛠 Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/gking432/my-token-launcher-standard.git
```

### 2️⃣ Navigate into the Project
```bash
cd my-token-launcher-standard
```

### 3️⃣ Install Dependencies
Make sure you have [Node.js](https://nodejs.org/) installed, then run:
```bash
npm install
```
_or_
```bash
yarn install
```

### 4️⃣ Run the Development Server
```bash
npm run dev
```
_or_
```bash
yarn dev
```

### 5️⃣ Open in Browser
Once running, visit:
```
http://localhost:3000
```
